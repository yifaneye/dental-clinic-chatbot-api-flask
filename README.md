# dental-clinic-chatbot-api
A hybrid chat bot, consisting of rule-based chat using RiveScript and ML+NLP using Wit.ai for intent processing (and APIs from dental-clinic-dentist-api and dental-clinic-timeslot-api)

The code is generated by:
```
swagger_py_codegen -s swagger.yaml chat --ui --spec
```

## Get started using Docker container

Build the Docker image:
```
docker build -t chatbot-api .
```

Run the Docker container:
```
docker run -p 8085:5003 -t chatbot-api __init__.py
```

The GUI of the API is at:
```
http://0.0.0.0:8085/static/swagger-ui/index.html#/
```
(Note: the exposed container port 5003 is mapped to the local machine port 8085)

Use the API from:
```
http://0.0.0.0:8085/v1/chat?message=hi
```
